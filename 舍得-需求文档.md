## 一、产品概述

### 1.1 产品定位
「舍得」是一款个人财务管理应用，核心聚焦于**应收/应付款项追踪**、**固定支出日历提醒**和**项目投入统计**，帮助用户清晰掌握资金流向与未来支出计划。

### 1.2 目标用户
- 有房贷、车贷、信用卡等固定月供的上班族
- 需要追踪借款往来的个人用户
- 有持续性投入项目（装修、学习、创业）的用户
- 希望直观了解未来支出计划的理财人群

### 1.3 核心价值
1. **清晰的账期管理**：一目了然的应收/应付时间线
2. **智能日历提醒**：不错过任何还款/收款日
3. **项目投入追踪**：掌握每个项目的累计投入
4. **预算控制**：月度预算管理，超支预警

### 2.3 添加记录模块

#### 表单字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 类型 | 单选 | ✅ | 应收款 / 应付款 |
| 分类 | 下拉 | ✅ | 应收：工资、投资、借款归还、其他<br>应付：网贷、信用卡、房贷、车贷、水电煤、其他 |
| 金额 | 数字输入 | ✅ | 支持小数点后两位 |
| 名称 | 文本输入 | ✅ | 账款名称（如：XX银行房贷） |
| 日期 | 日期选择 | ✅ | 首次到期日 |
| 周期 | 下拉 | ✅ | 一次性 / 每月 / 每周 / 每年 / 自定义 |
| 自定义周期 | 数字输入 | 条件 | 输入天数（如：每15天） |
| 关联项目 | 下拉 | ❌ | 可选，关联到已有项目 |
| 备注 | 多行文本 | ❌ | 补充说明 |

#### 表单交互
- 类型切换时，分类选项联动更新
- 选择"自定义"周期时，显示天数输入框
- 提交成功后显示 Toast 提示，表单重置
- 支持保存为草稿（意外退出不丢失）

**日期标记规则：**
| 标记 | 含义 |
|------|------|
| 灰色圆点 `●` | 有款项，全部未处理 |
| 绿色圆点 `●` | 有款项，全部已处理 |
| 红色圆点 `●` | 有逾期未处理款项 |
| 无标记 | 当日无款项 |

#### 2.4.2 展开状态 - 周视图 + 详情面板

**交互流程：**
1. 用户点击某日期
2. 月视图收缩为周视图（仅显示当前周）
3. 详情面板从上方滑入

```
┌─────────────────────────────────────┐
│                                     │
│   ┌─────────────────────────────┐   │
│   │  2026年6月15日 周一          │   │

│   │  🔴 房贷还款                 │   │
│   │     ¥8,500.00               │   │
│   │     周期：每月               │   │
│   │     状态：待支付             │   │
│   │                             │   │
│   │  ┌───────────────────────┐  │   │
│   │  │    ✅ 标记为已支付      │  │   │
│   │  └───────────────────────┘  │   │
│   │                             │   │
│   │  🔴 信用卡还款               │   │
│   │     ¥3,200.00               │   │
│   │     ...                     │   │
│   └─────────────────────────────┘   │
│                                     │
├─────────────────────────────────────┤
│       ←  6月第3周 (6/15-6/21)  →   │
│  一   二   三   四   五   六   日   │
│  15   16   17   18   19   20   21  │
│  [●]                                │  ← [●] 选中态
└─────────────────────────────────────┘
```

**交互细节：**
| 操作 | 效果 |
|------|------|
| 点击日期 | 月历→周历收缩 + 详情面板滑入 |
| 左右滑动周历 | 切换上/下周，详情随之更新 |
| 点击周历其他日期 | 切换详情，选中态跟随 |
| 点击遮罩/下滑手势 | 详情消失，周历恢复月历 |

**动效参数：**
| 阶段 | 时长 | 缓动 |
|------|------|------|
| 月历→周历 | 350ms | ease-out |
| 详情面板滑入 | 300ms | 弹性缓动，延迟100ms |
| 详情面板滑出 | 200ms | ease-in |
| 周历→月历 | 350ms | ease-in-out |

## 三、非功能需求

### 3.1 性能要求
- 首页加载时间 < 1秒
- 页面切换动画流畅 60fps
- 列表滑动无卡顿（支持1000+条目）

### 3.2 数据安全
- 云同步数据加密传输（HTTPS）
- 本地敏感数据加密存储
- 支持数据导出备份

### 3.3 兼容性
- iOS 15.0+
- Android 10.0+（API 29+）
- 适配 iPhone SE ~ iPhone 15 Pro Max
- 适配主流 Android 分辨率

### 3.4 可访问性
- 支持系统字体大小调节
- 关键信息不仅依赖颜色区分（添加图标/文字）
- 屏幕阅读器基本支持

## 五、交互规范

### 5.1 手势操作
| 手势 | 场景 | 效果 |
|------|------|------|
| 左右滑动 | 日历周视图 | 切换周 |

### 5.2 反馈机制
| 操作 | 反馈 |
|------|------|
| 按钮点击 | 触觉反馈（轻量） |
| 标记已支付 | Toast + 触觉反馈 |

| 超出预算 | 警告对话框 + 推送通知 |

- 统计无数据：显示空状态插图 + 引导文案
- 项目列表为空：显示"创建第一个项目"
