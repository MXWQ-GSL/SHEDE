> 版本：v1.0 | 最后更新：2026-06-02

## 一、设计理念

### 美学方向：Luxury Refined（奢华精致）

「舍得」定位为高端个人财务管理工具，美学方向选择 **Luxury Refined**——克制的奢华、精致的细节、呼吸感的留白。

**核心关键词：**
- **克制的奢华**：不堆砌装饰，用材质和光影体现质感
- **数字优先**：金额是视觉主角，界面为数字服务
- **呼吸感**：大量留白，信息密度适中
- **触感真实**：玻璃质感、微妙光影、细腻动效

**记忆点：**
- 首页净资产数字的**渐变光晕效果**（从数字中心向四周扩散的微妙光晕）
- 卡片的**磨砂玻璃质感**（backdrop-filter: blur）
- 日历面板滑入时的**弹性动效**
- 金额变化时的**数字滚动动画**

### 设计原则
1. **信息优先**：数字和金额是主角，界面为信息服务
2. **减少认知负担**：每个页面只有一个核心任务
3. **一致性**：相同功能使用相同交互模式
4. **可预测性**：用户能预知操作结果
5. **质感优先**：宁可少一个功能，不可少一个细节

## 三、字体系统

### 字体选择

**风格定位：** 克制、精致、有记忆点

**英文/数字（金额展示）：**
- **首选**：`DM Sans`（现代几何风格，数字设计精致）
- **备用**：`SF Pro Display`（iOS）/ `Google Sans`（Android）
- **金额专用**：`JetBrains Mono` 或 `DM Mono`（等宽，tabular-nums）

**中文：**
- **首选**：`HarmonyOS Sans SC`（华为鸿蒙字体，现代精致）
- **备用**：`PingFang SC`（iOS）/ `Noto Sans SC`（Android）
- **标题可选**：`ZCOOL XiaoWei`（站酷小薇，用于品牌标题点缀）

**字体加载策略：**
- 关键字体 preload：DM Sans (400, 600, 700)、HarmonyOS Sans SC (400, 600)
- font-display: swap 避免 FOIT
- 系统字体作为 fallback

### 字号规范

| 层级 | 名称 | 大小 | 行高 | 字重 | 用途 |
|------|------|------|------|------|------|
| Display | 展示 | 34px | 1.2 | Bold | 首页金额大数字 |
| Title1 | 标题1 | 28px | 1.3 | Semibold | 页面标题 |
| Title2 | 标题2 | 22px | 1.3 | Semibold | 区块标题 |
| Title3 | 标题3 | 20px | 1.4 | Semibold | 卡片标题 |
| Headline | 标题 | 17px | 1.4 | Semibold | 列表项标题 |
| Body | 正文 | 17px | 1.5 | Regular | 正文内容 |
| Callout | 引用 | 16px | 1.5 | Regular | 辅助说明 |
| Subhead | 副标题 | 15px | 1.5 | Regular | 次要信息 |
| Footnote | 脚注 | 13px | 1.5 | Regular | 时间戳、标签 |
| Caption1 | 说明1 | 12px | 1.5 | Regular | 辅助文字 |
| Caption2 | 说明2 | 11px | 1.5 | Regular | 最小文字 |

### 金额数字特殊处理
- 使用等宽字体（SF Mono、Roboto Mono）或 `font-variant-numeric: tabular-nums`
- 金额前缀 `¥` 使用较小字号（0.75em）
- 大金额添加千分位分隔符：`¥12,500.00`

### 材质系统（核心差异化）

**磨砂玻璃卡片（Glass Card）：**
```css
.glass-card {
  background: rgba(20, 20, 25, 0.8);
  backdrop-filter: blur(20px) saturate(1.8);
  -webkit-backdrop-filter: blur(20px) saturate(1.8);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 16px;
}
```

**光晕效果（用于强调数字）：**
```css
.glow-text {
  text-shadow: 
    0 0 20px rgba(10, 132, 255, 0.3),
    0 0 40px rgba(10, 132, 255, 0.1);
}
```

## 七、动效规范

### 时长标准

| 类型 | 时长 | 用途 |
|------|------|------|
| 微交互 | 150-200ms | 按钮点击、开关切换 |
| 状态变化 | 200-300ms | 面板展开/收起、列表项展开 |
| 页面转场 | 300-400ms | 日历状态切换、详情弹出 |
| 复杂动画 | 400-500ms | 多元素编排动效 |

### 缓动曲线

| 类型 | 曲线 | 用途 |
|------|------|------|
| 标准 | `cubic-bezier(0.25, 0.1, 0.25, 1)` | 通用过渡 |
| 弹性 | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 面板弹出 |
| 减速 | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入 |
| 加速 | `cubic-bezier(0.4, 0, 1, 1)` | 元素退出 |

### 日历模块动效细节

```
【展开详情面板】
1. 月历 → 周历收缩
   - 高度动画：350ms，ease-out
   - 保留当前周，其余行折叠隐藏
   
2. 详情面板滑入
   - 从日历区域上方滑入
   - 时长：300ms，弹性缓动
   - 延迟：100ms（与日历收缩错开）

【收起详情面板】
1. 详情面板滑出
   - 时长：200ms（退出比进入快）
   
2. 周历 → 月历展开
   - 时长：350ms，ease-in-out
```

### 减少动效支持
- 监测系统 `prefers-reduced-motion` 设置
- 开启时：移除弹性效果，缩短时长至 100-150ms

### 特色动效

**数字滚动动画（金额变化时）：**
- 每位数字独立滚动
- 时长：600ms，带惯性
- 缓动：`cubic-bezier(0.16, 1, 0.3, 1)`

**卡片入场动画：**
- 从下方滑入 + 透明度渐变
- 多卡片时 stagger 50ms
- 时长：400ms

**状态切换反馈：**
- 「已支付」按钮：按下缩放0.95 → 弹回1.0 + 触觉反馈
- 日历日期选中：背景色渐变 + 微弹效果

**背景氛围动效：**
- 首页背景：缓慢移动的渐变光斑（60s循环）
- 深色模式：深蓝 → 深紫渐变，opacity 5-10%

## 十、深色/浅色模式切换

### 切换规则
- 跟随系统设置（默认选项）
- 支持手动切换为浅色或深色
- 入口位置：我的 → 设置 → 主题设置
- 切换动画：渐变过渡 300ms
- 首页不放主题切换入口

### 需特别处理的元素
- 图标：深色模式下降低饱和度
- 图表：深色模式下降低背景网格对比度
- 图片：深色模式下添加 10% 白色遮罩（可选）
- 分割线：深色模式下使用边框替代阴影

## 十一、视觉氛围与质感

### 背景设计

**深色模式：**
- 主背景：纯黑偏蓝 `#0A0A0F`
- 添加微妙渐变：`linear-gradient(135deg, #0A0A1A 0%, #0A0A0F 50%, #0F0A1A 100%)
- 可选：极低透明度的几何纹理（0.5% opacity）

**浅色模式：**
- 主背景：柔和灰 `#F2F2F7`
- 卡片：纯白 `#FFFFFF`
- 可选：细腻的纸张纹理（0.3% opacity）

### 卡片质感细节

**磨砂玻璃效果参数：**
- 模糊度：20-30px
- 饱和度：1.5-2.0
- 边框：1px solid rgba(255,255,255,0.08)（深色）/ rgba(0,0,0,0.05)（浅色）
- 阴影：根据层级使用不同强度

**金额数字特殊处理：**
- 大金额（首页净资产）：使用渐变色
  ```css
  .amount-hero {
    background: linear-gradient(135deg, #F5F5F7 0%, #8E8E93 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
  }
  ```
- 应收金额：绿色系渐变
- 应付金额：红色系渐变

### 分割线处理

**深色模式：**
- 使用 1px 边框而非阴影
- 颜色：`rgba(255,255,255,0.06)`
- 可选：渐隐分割线（左右透明，中间显色）

**浅色模式：**
- 使用极浅阴影或边框
- 颜色：`rgba(0,0,0,0.05)`
